# 背景音乐文件（必放）

**不放置以下文件时，背景音乐会报错或静默失败。**

请将这两个 MP3 文件放在本目录（`client/public/`）下：

| 文件名        | 用途 |
|---------------|------|
| **bgm.mp3**   | 关卡内（游戏中）背景音乐 |
| **all-bgm.mp3** | 主题选择、关卡选择、图鉴等页面统一背景音乐 |

- 放好后刷新页面即可，无需重启。
- 游戏外 BGM：进入首页后点击 **「点击播放背景音乐」** 按钮，或先点击页面任意处再播。
- 关卡内 BGM：在用户点击选关时自动尝试播放；若失败，请检查 `client/public/bgm.mp3` 是否存在。
- 若控制台出现 `[puzzleBgm]` / `[allBgm]` 的报错，多为文件缺失或路径不对，请确认上述两个文件在 `client/public/` 下。
